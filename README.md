# FactuFácil 🚀

**FactuFácil** es una aplicación de facturación moderna, rápida y sencilla diseñada específicamente para autónomos y pequeñas empresas. 

## ✨ Características principales

- 📄 **Gestión de Facturas:** Crea, edita y organiza tus facturas de forma profesional.
- 📋 **Presupuestos:** Genera presupuestos detallados y conviértelos en facturas con un solo clic.
- 👥 **Gestión de Clientes:** Mantén una base de datos de tus clientes para agilizar el proceso de facturación.
- 🖨️ **Impresión y PDF:** Genera documentos listos para enviar o imprimir directamente desde el servidor (evita problemas de permisos).
- 📱 **Diseño Responsive:** Optimizado para móviles y tablets con protecciones contra conexiones lentas.
- ☁️ **Sincronización en la nube:** Integración total con Firebase (Auth & Firestore).
- 📶 **Modo Offline:** Persistencia de datos habilitada para trabajar en zonas con mala cobertura.

## 🛠️ Tecnologías utilizadas

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Base de Datos y Auth:** [Firebase](https://firebase.google.com/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) + [Shadcn UI](https://ui.shadcn.com/)
- **IA:** [Genkit](https://github.com/firebase/genkit)
