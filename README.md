# FactuFácil 🚀

**FactuFácil** es una aplicación de facturación moderna, rápida y sencilla diseñada específicamente para autónomos y pequeñas empresas. 

## ✨ Características principales

- 📄 **Gestión de Facturas:** Crea, edita y organiza tus facturas de forma profesional.
- 📋 **Presupuestos:** Genera presupuestos detallados y conviértelos en facturas con un solo clic.
- 👥 **Gestión de Clientes:** Mantén una base de datos de tus clientes para agilizar el proceso de facturación.
- 🖨️ **Impresión y PDF:** Genera documentos listos para enviar o imprimir directamente desde el navegador.
- 📱 **Diseño Responsive:** Accede y gestiona tu negocio desde cualquier dispositivo (móvil, tablet o PC).
- ☁️ **Sincronización en la nube:** Tus datos están seguros y accesibles gracias a la integración con Firebase.
- 📶 **Modo Offline:** Sigue trabajando incluso sin conexión a internet.

## 🛠️ Tecnologías utilizadas

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Base de Datos y Auth:** [Firebase](https://firebase.google.com/) (Firestore & Authentication)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Componentes UI:** [Shadcn UI](https://ui.shadcn.com/)
- **Iconos:** [Lucide React](https://lucide.dev/)
